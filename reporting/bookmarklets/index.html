Bookmarlet information

Bookmarklets are pieces of javascript that are ran as a bookmark,
some popular bookmarklets are below. These are hosted on
this site for the use of reporting them and will not be
shared. To use these bookmarklets, drag them to your
bookmarks, for the web page editing you can just click
it on any page to enable editing. On the disable extensions
bookmarklet, you must click it once on this site, once you
are sent to the chrome webstore you click it again and you
will be presented with toggles with all extensions visible.
The scripts have an alert window with a disclaimer, just
click OK to continue running the script.

  • Enable page editing
  • Modified CompactCow disable extensions
The disable extensions bookmarklet is especially dangerous as it
allows disabling of Goguardian and the other filtering plugins.
This particular bookmarlet is already being looked at by
Goguardian and Google respectively.

The disable extension script is made by CompactCow and modified
to add the disclaimer, the original instructions can be found
on their site here this is blocked by default but anyone can
just put the script on another site as seen here. The full
script source can be found here and mine can be found here.
along with that Ingot is a similar script just wtih a nicer
user interface, but seems that GG has been successful in
restricting the webstore url they use while Compactcow
is using a randomly generated one looking at the script.
Lastly the web editing script can be found here. The
bookmarklet above just has the script as a one liner
instead of a fetch due to CORS restrictions in place by
most sites.

Currently theres no direct fix for this, only partial fixes
one that will fix one liner bookmarklets such as the web
page editing one can be done by taking the following steps
in Google Admin Console:
  1. Sign in to Google Admin Console
  2. Go to Devices > Chrome > Settings.
  3. Select the OU group the students are in on the left
  4. Scroll to URL blocking.
  5. Add the line javascript://*

There is currently one fix when it comes to bookmarklets
which fetch their script such as the one to disable
extensions, this is to use chrome v106 as google has
implemented a fix to the management api the chrome
webstore uses rending these scripts useless.
